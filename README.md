# Sign-Language-detection-system
The Sign Language Detection System is an AI-based project that converts hand gestures into text in real time using computer vision and machine learning. It helps bridge communication gaps for deaf and mute individuals by enabling easy interaction and promotes inclusivity through practical technology.It can detect Alphabets , numbers from 1 to 10 , Thank you please words. 
We used a venv environment in this to make it adapt to the environment 
✨ Features
Real-time hand gesture detection
Webcam-based input
Sign language recognition
Fast and accurate predictions
User-friendly interface
Supports multiple gestures/alphabets
Lightweight and easy to run locally
🛠️ Technologies Used
Python
OpenCV
NumPy
MediaPipe
